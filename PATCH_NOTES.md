# SchoolHub Patch Notes

## v1.0.0 - SchoolHub 첫 정식 배포

SchoolHub v1.0.0은 학교 커뮤니티 운영을 위한 기본 기능을 완성한 첫 정식 버전입니다.

---

## 추가된 기능

### 학교 커뮤니티

- 학교 선택 페이지 추가
- 학교별 커뮤니티 메인 페이지 추가
- 학교 메인에서 게시판, 분실물, 공지사항 접근 가능

### 게시판

- 게시글 목록 조회 기능 추가
- 게시글 작성 기능 추가
- 게시글 카테고리 기능 추가
  - 자유
  - 질문
  - 건의
- 게시글 제목, 내용, 작성자 검색 기능 추가
- 게시글 상세 페이지 추가
- 게시글 수정 기능 추가
- 게시글 삭제 기능 추가
- 게시글 작성 시 관리자 승인 후 공개되도록 변경
- 삭제되었거나 존재하지 않는 게시글 접근 시 404 페이지 표시

### 댓글

- 게시글 상세 페이지에 댓글 작성 기능 추가
- 댓글은 관리자 승인 후 공개되도록 설정
- 관리자 댓글 승인 및 삭제 기능 추가

### 공지사항

- 공지사항 목록 및 상세 기능 추가
- 관리자 공지사항 작성 기능 추가
- 관리자 공지사항 수정 기능 추가
- 관리자 공지사항 삭제 기능 추가
- 고정 공지 기능 추가
- 학교 메인에 공지사항 패널 추가

### 건의사항

- 사용자 건의사항 제출 기능 추가
- 관리자 건의사항 확인 기능 추가
- 관리자 건의사항 삭제 기능 추가

### 분실물

- 분실물 목록 조회 기능 추가
- 분실물 등록 기능 추가
- 분실물 상세 페이지 추가
- 발견 장소, 찾아갈 장소 입력 기능 추가
- 분실물 검색 기능 추가
- 분실물은 관리자 승인 후 공개되도록 설정
- 관리자 분실물 승인 기능 추가
- 관리자 분실물 삭제 기능 추가

### 관리자 페이지

- 관리자 로그인 기능 추가
- 관리자 대시보드 추가
- 승인 대기 게시글 카드 추가
- 승인 대기 댓글 카드 추가
- 승인 대기 분실물 카드 추가
- 공지사항 관리 카드 추가
- 건의사항 관리 카드 추가
- 관리자 전용 게시판 관리 페이지 추가
- 관리자 전용 분실물 관리 페이지 추가
- 관리자 전용 header/footer 추가
- 관리자 페이지 UI 개선

### UI / 디자인

- 사용자 공통 header 추가
- 사용자 공통 footer 추가
- 관리자 공통 header 추가
- 관리자 공통 footer 추가
- 사용자 header/footer 색상 통일
- 짧은 페이지에서도 footer가 아래에 위치하도록 수정
- 관리자 상세 페이지 메타 정보 UI 개선
- 삭제 버튼 클릭 시 확인창 추가
- 404 페이지 추가

### 배포

- Render 배포 설정 적용
- PostgreSQL 연결 설정 적용
- 배포 환경용 `DATABASE_URL` 지원
- 서버 시작 시 DB 테이블 자동 생성 스크립트 추가

---

## 수정된 사항

- 분실물 상태값인 `찾는 중`, `찾음` 표시 제거
- 분실물 관리는 다른 게시물처럼 승인 중심 구조로 변경
- 관리자 하단 관리 페이지에서는 승인보다 삭제 관리 중심으로 변경
- 일반 사용자 화면에는 관리자 페이지 링크가 노출되지 않도록 유지

---

## 알려진 사항

- 관리자 세션은 현재 기본 MemoryStore를 사용합니다.
- v1.0에서는 단일 학교 운영을 기준으로 구성되어 있습니다.
- 추후 여러 학교 확장, 관리자 권한 세분화, 세션 저장소 개선 등을 진행할 수 있습니다.
